## About Hangar Bag

Hangar Bag is the first tool organizer designed to mount directly to your airplane during maintenance and inspection. Keep your tools within arm's reach, off the ground, and out of the way — reducing FOD risk and saving time on every job.

### Key Features

- **Instant Attachment** — Clips onto wing struts, landing gear, engine mounts, and more
- **Multi-Pocket Organization** — Purpose-built compartments for aviation tools
- **Aircraft-Safe Materials** — Non-abrasive, non-conductive, avionics-friendly
- **Lightweight & Durable** — Built for daily hangar use without adding weight
- **Reduces FOD Risk** — Tools stay secured to the aircraft, not scattered on the floor
- **Universal Fit** — Adjustable mounting for single-engine pistons to light twins

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/hangar-bag.git
cd hangar-bag
```

### 2. Open locally

Just open `index.html` in your browser — no server needed.

### 3. Deploy to GitHub Pages

1. Go to your repo on GitHub
2. Navigate to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch and **/ (root)** folder
5. Click **Save**
6. Your site will be live at `https://YOUR_USERNAME.github.io/hangar-bag/`
